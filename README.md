
🚀 Share Your Game with the World

Ready to show off what you built? Follow these steps and let your game shine:

🎥 1. Capture & Upload Your Gameplay

Record a short gameplay video that highlights your game in action—show the core mechanics, features, and overall experience.

Then:
	•	Fork this repository to your GitHub account
	•	Upload your gameplay video to your forked repository
	•	Add the video to your README file (you can embed it or include a link)

💻 2. Upload Your Code

Push your full game project to your own GitHub repository.
Make sure your project includes:
	•	Well-organized code
	•	All necessary assets
	•	A clear README explaining your game and how to run it

🌍 3. Join the Challenge

Take a screenshot of your game and share it on social media using:
#تحديات_مسار

Let others see your creativity, get inspired, and be part of the community!
